
## Описание проекта

Проект представляет собой шаблон резюме, где текстовые элементы можно произвольно модифицировать. По окончанию редактирования текстового блока, срабатывает анимация. Все измененные данные сохраняются при обновлении страницы. Реализован функционал скачивания резюме в формате pdf при нажатии на кнопку "Download as PDF".

## Инструкции:

Для взаимодействия с веб-страницей достаточно просто открыть файл index.html в веб-браузере.

## Dev-Инструкции:

Для работы с самим проектом необходимо установить зависимости командой:
```
npm ci
```
